新建系统变量

右击“此电脑”，依次点击“属性”，“高级系统设置”，“环境变量”，在“系统变量”处新建以下变量：

变量名 变量值
JAVA_HOME jdk的位置，如D:\Java\jdk-14
CLASSPATH .;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;

jdk1.5版本以上不需要设置CLASSPATH变量
